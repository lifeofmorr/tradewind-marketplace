# Post-Demo Follow-Up Templates

**Owner:** Don Morrison
**Date:** 2026-05-27
**Purpose:** Short, human follow-ups after a 10-minute demo or partner call.

Every template is copy-paste ready. Edit `[FIRST]`, `[COMPANY]`, and the one
specific detail you remember from the call. Don't edit the tone.

Send all of these from `don@lifeofmorr.com`, same thread as the original
reply. Founder voice. No marketing CTAs.

## Universal rules for these follow-ups

- Always send from `don@lifeofmorr.com`. No team alias.
- Always reply in-thread when possible — never start a fresh thread
  unless it's a brand-new topic.
- Always reference one specific detail from the prior conversation.
- Never send the same template twice to the same person.
- Never use exclamation points except in greetings.
- Never include marketing footers, social links, or unsubscribe banners
  that don't belong on a 1:1 founder email.
- Log every follow-up in `outreach_messages` (direction=outbound,
  channel=email) so the CRM knows what's been sent.

If a follow-up doesn't get a response in 5–7 business days, send one
gentle nudge ("just bumping this in case it got buried"), then move
the lead to `follow_up_later` and stop.
