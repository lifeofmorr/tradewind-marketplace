# Archive

Point-in-time generated reports: audits, launch/beta checklists, status
reports, QA snapshots, outreach planning, and "readiness" documents produced
during the 2025–2026 build-up and beta push. They are kept for historical
reference only.

**Nothing in this directory is maintained.** Claims, scores, and checklists in
these files reflect the moment they were written and are very likely stale.

For current, maintained documentation see:

- [`../../README.md`](../../README.md) — project overview and quick start
- [`../../SETUP.md`](../../SETUP.md) / [`../../DEPLOY.md`](../../DEPLOY.md) — setup and deployment
- [`../`](../) (`docs/`) — durable reference docs (runbooks, security
  architecture, design system, pricing, audit score, etc.)
